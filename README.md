# ⏰ Study Reminder App (Python)

This is a simple Study Reminder application built using Python.  
The program sends a desktop notification and gives a voice reminder at specified times.

It was created as a beginner project to practice Python modules and automation concepts.

## 🚀 Features

- Shows desktop notification
- Speaks reminder using text-to-speech
- Custom reminder times
- Prevents duplicate reminders

## 🛠 Technologies Used

- Python
- time module
- pyttsx3 (Text-to-Speech)
- plyer (Desktop Notification)

## 📌 How It Works

The program checks the current system time continuously.
When the time matches the reminder time:
- It shows a notification
- It plays a voice reminder
- It avoids repeating the same reminder again

## 🎯 Purpose

This project was built for learning purposes and to improve productivity by setting study reminders.
